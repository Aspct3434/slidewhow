# slidewhow

An interactive slideshow website that displays images with descriptions. Features include:

- Automatic and manual navigation
- Keyboard controls and pause on hover
- Descriptive captions for each image

## Running locally

Open `index.html` directly in your browser or serve the folder using a simple server:

```bash
python3 -m http.server
```

Then navigate to [http://localhost:8000](http://localhost:8000) in your browser.
